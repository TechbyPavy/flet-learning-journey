# Aula 01 — Introdução ao Flet 

Baseado no vídeo [Aula 1 - Flet do Zero ao Avançado](https://www.youtube.com/watch?v=IjQry9Pnvgw) 
do canal [PyFlet](https://www.youtube.com/@pyflet). 

## O que aprendi 

- Instalar Python e VSCode
- Criar o primeiro ficheiro com Flet
- Executar uma app simples com `ft.app(target=main)`

## Código

O ficheiro `main.py` contém o código da primeira aplicação criada com Flet: 
- Importação do módulo Flet
- Criação da função `main`
- Adição de um texto simples à página
- Execução da app com `ft.app(target=main)`

## Impressões pessoais 

Esta foi a minha primeira experiência com Flet. Gostei da ideia de criar interfaces só com Python.
