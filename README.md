# Flet learning Journey

## Objetivo

Documentar cada aula, projeto e experiência prática enquanto aprendo a criar aplicações com Flet.

## Fonte de estudo

Os conteúdos são baseados principalmente nos vídeos do canal [PyFlet](https://www.youtube.com/@pyflet), especialmente a série "Flet do Zero ao Avançado", com créditos ao criador Jonas Franco.  
As notas aqui escritas são interpretações e resumos pessoais, não cópias diretas do conteúdo original.
